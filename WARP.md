# Warp AI 优化配置

## 项目简介
这是一个电影推荐相关的资源仓库，主要包含影视推荐、观影指南和娱乐资源分享。

## 常用命令
```bash
# 查看最近更新的文档
ls -lt *.md | head -5

# 搜索特定内容
grep -r "关键词" *.md
```

## Git 工作流
```bash
# 添加新文档
git add 202508.md
git commit -m "Add 202508 movie recommendations"
git push origin main
```

## 文档约定
- 文档命名格式：YYYYMM.md
- 内容包含电影推荐、观影指南、娱乐资讯
